Then,

  1. Check your python version.
  2. Run python in any terminal to know your python version.
  3. You will get some kind of result: Python 3.9.7 (tags/v3.9.7:1016ef3, Aug 30 2021, 20:19:38) [MSC v.1929 64 bit (AMD64)] on win32
  4. Here my python version is 3.9.7 with 64 bit (AMD) architecture.
  5. Go to (Unofficial Windows Binaries for Python) : https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio
  6. Download correct version of wheel. For me it is: PyAudio‑0.2.11‑cp39‑cp39‑win_amd64.whl (39 is python version & amd64 is architecture.)
  7. Open the terminal where you downloaded the Python wheel. (For example: C:\Users\john\Downloads>)
# Instead of "PyAudio‑0.2.11‑cp39‑cp39‑win_amd64.whl" write your wheel name and run command.
pip install PyAudio‑0.2.11‑cp39‑cp39‑win_amd64.whl

# If you are getting this message, then PyAudio is Successfully installed.
