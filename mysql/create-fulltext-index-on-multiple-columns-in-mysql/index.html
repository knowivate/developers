Create FULLTEXT index on multiple columns in MySQL?

Here we will create FULLTEXT index of three columns.

ALTER TABLE `database_name`.`table_name` ADD FULLTEXT (`column1`, `column2`, `column3`);

# Here
# `database_name` = The name of your database.
# `table_name` = The name of your table.

# column1 = Name of column.
# column2 = Name of column.
# column3 = Name of column.
