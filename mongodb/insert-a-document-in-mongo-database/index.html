Insert a document in MongoDb

# Insert a document in collection.
# Here we are inserting four key and values.

db.collectionName.insert({
    "key_name": "value_here",
    "ke2": "value2",
    "store_number": 5,
    "database": "MongoDb"
})

Explanation:

# db = name of current database.
# collectionName = collection name of current database. 
# insert = MongoDb command to insert a document in collection.
# "key_name" = key to insert.
# "value_here" = value of key.
